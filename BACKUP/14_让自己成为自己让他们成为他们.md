# [让自己成为自己让他们成为他们](https://github.com/Quitino/gitblog/issues/14)

昨晚看电视的时候恍惚间看见这几句台词，有点共情，倒回去又看了一遍，今天不是特别忙索性记录一下。简单截了几张图丢给ai帮忙拼接一下：

```
请将以上十五张图进行电影截图纵向拼贴：
- 第一张截图在最上方，完整展示
- 后续每张依次往下叠放
- 每一层只露出下面的台词区域 / 字幕部分
- 所有字幕区域裁剪为相同高度，视觉间距一致
```

打开编辑窗口准备记录的时候，内心想这两句有点口语化，要不要整文雅点，然后在脑海中搜索了一下,，蹦出了一句“让花成花”，于是又搜索了一下完整句子，搜索出来是这样的（标注是 “杨绛说”）：

> 无论什么关系，情分被消耗殆尽，缘分便走到了终点，没有谁对谁错，最后所有的错都应该归咎在自己身上，然后礼貌地退场。把自己还给自己，把别人还给别人，让花成花，让树成树。从此，山水一程，再不相逢，愿来生，不见，不欠，不念。

给我的感觉是讲感情的，太狭隘，也不符合我昨晚的共情感受；然后我又想这怎么看也不像一个长者的文字啊，更像是中学生的qq微信签名啊；于是我想查杨绛原话是怎么说的，结果人家根本没说过这句话...都是自媒体为了流量，把这些句子扣杨绛、三毛头上。emmm，我为什么会蹦出“让花成花”，书读少了蹦不出其他的...

最后我还是选择使用“让自己成为自己让他们成为他们”，更接地气更烟火更中性；这里我想到的是对双方都有益的一种思维方式，而不是单方面的简单的情感宣泄情感疗伤啥的。没有自我批评也没有批评别人，没有什么放下执念走出困境，就是日常和人和事相处的一种思维，我认为是一种中性的表达，没有阴阳没有好恶。

比较词穷，说不出什么感悟，但就是有感悟hh。
