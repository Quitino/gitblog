# [北京中轴线citywalk](https://github.com/Quitino/gitblog/issues/8)


2026.04.26.周末反穿了中轴线，从森林公园南门出发，原计划走到永定门。实际情况从森林公园南门出发途径鸟巢、水立方、中轴线北广场、钟楼、鼓楼、烟袋斜街、什刹海、景山公园、故宫博物院、中山公园、天安门广场（需要提前一天预约，没进；捡了一个带花的梧桐树枝别在背包上，被告知不让带鲜花进天安门广场）、国家大剧院、纪念堂，然后时间比较晚了，过了纪念堂就骑上单车到国贸附近吃晚饭了。全程 26km，徒步20km，汽车6km. 午饭和晚饭都是吃的川菜馆。

早上出门时天气是阴着的，结果刚开始走没多久就开始出太阳了，被迫买了帽子，有太阳的徒步不带墨镜走到下半程也会有点头晕恍惚。周末奥森人不是特别多，到了钟楼鼓楼到天安门那段核心旅游区人就多起来了，也有较多的境外游客，还是挺好玩的，值得二刷，天坛地坛还没刷到。
